# RiskLens – Area-Based Crime Surveillance System 🚨

A web application for tracking and analyzing crime incidents by area, identifying patterns, and visualizing data to support surveillance and decision-making.

## Project Overview
RiskLens helps law enforcement and city planners monitor crime activity effectively. The system allows users to log incidents, view area-wise crime statistics, and detect trends through interactive dashboards.
